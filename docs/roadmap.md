# Roadmap
- MVP: core engine loop, module registry, sample plugin, minimal example.
- Scripting: real Lua VM integration, script binding helpers, error reporting.
- Visual blocks: GUI editor with node graph, block palette, code generation hooks.
- Plugins: platform-specific dynamic loader, hot-reload, signing/versioning.
- Tooling: CI, clang-format/clang-tidy, packaging scripts, docs site.
